- Copyparty is a portable and self-hosted file server written in Python. It allows users to share files through a web interface with features like media streaming, file uploads, and WebDAV support. The tool is designed with security in mind, offering robust permission management and the ability to run on various platforms, including via Docker. [(read in 2 min)](https://github.com/9001/copyparty?tab=readme-ov-file)
- Multiplex is a lightweight and straightforward terminal multiplexer written in the Go programming language. It serves as a full-featured alternative to more complex tools like tmux and screen. The project aims to provide essential multiplexing capabilities such as managing panes, windows, and sessions with a focus on simplicity. [(read in 2 min)](https://github.com/sebastien/multiplex)
