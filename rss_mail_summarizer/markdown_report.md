## GitHub

- This repository hosts the source code for carbonOS, an independent Linux distribution designed for robustness and user-friendliness. It utilizes a read-only root filesystem and relies on Flatpak for applications to ensure system stability and security. The distribution features atomic updates, meaning updates are either fully applied or not at all, preventing a broken system state. [(read in 4 min)](https://github.com/crbnos/carbon)
