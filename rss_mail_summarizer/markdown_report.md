- The Fire-Flyer File System (3FS) is a high-performance distributed file system designed for AI training and inference workloads, utilizing SSDs and RDMA networks. It offers features like a disaggregated architecture, strong consistency, and file interfaces, supporting diverse workloads such as data preparation, dataloaders, checkpointing, and KVCache for inference. The system aims to simplify the development of distributed applications by providing a shared storage layer. [(click here)](https://github.com/deepseek-ai/3FS)
- Superglue is an open-source data connector designed to simplify integration with complex or legacy APIs by automatically generating API configurations, handling pagination and authentication, and transforming data into a desired schema. It validates data, fixes transformations, and supports various data sources and authentication methods. The tool aims to reduce the time spent on connecting to APIs and ensuring data integrity. [(click here)](https://github.com/superglue-ai/superglue)
- Microsoft has confirmed that Skype will be discontinued on May 5, 2025, after more than 21 years of service. The company plans to migrate Skype users to Microsoft Teams, allowing them to log in with their Skype credentials and access existing contacts and chats. Users who prefer not to migrate can export their Skype data, including contacts, call history, and chats. [(click here)](https://arstechnica.com/gadgets/2025/02/on-may-5-microsofts-skype-will-shut-down-for-good/)
- Ladybird is an independent web browser under development, utilizing a novel engine based on web standards and a multi-process architecture for robustness. It reuses core library support components from SerenityOS, including LibWeb, LibJS, and LibWasm. The browser is currently in a pre-alpha state, intended for developer use, and runs on Linux, macOS, Windows (with WSL2), and other *Nixes. [(click here)](https://github.com/LadybirdBrowser/ladybird)
