# News of the Day

## GitHub

- This repository contains an explainer for a proposed web platform feature that would allow rendering arbitrary HTML content directly into a `<canvas>` element. The proposal, from the Web Incubator Community Group (WICG), introduces a new `drawHTML` method to the canvas rendering context. This feature aims to simplify the process of drawing rich text and complex layouts onto a canvas, which is currently a difficult task. [(read in 9 min)](https://github.com/WICG/html-in-canvas)
