## GitHub

- Jan is an open-source, local-first AI assistant designed to run large language models (LLMs) offline on various operating systems, prioritizing user privacy and speed. The project emphasizes extensibility, allowing users to customize models and integrate with different AI frameworks like Llama.cpp and TensorRT-LLM. Jan aims to provide a powerful, secure, and customizable local AI experience for desktop users. [(read in 6 min)](https://github.com/menloresearch/jan)
- POML (Privacy-Oriented Machine Learning) is a framework developed by Microsoft for building machine learning models with strong privacy guarantees. It focuses on implementing techniques such as differential privacy and federated learning to enable secure data analysis and model training without compromising sensitive information. The repository provides tools and examples to help developers integrate privacy-preserving AI solutions into their projects. [(read in 3 min)](https://github.com/microsoft/poml)
- Poltergeist is a Swift library designed to assist iOS developers in detecting and preventing memory leaks within their applications. It helps identify common memory management issues, such as retain cycles, to improve app stability and performance. The library integrates with Xcode, providing clear diagnostics and tools for efficient debugging of memory-related problems. [(read in 2 min)](https://github.com/steipete/poltergeist)
