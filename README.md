# RSS-Mail-Summarizer

Aktuelle Projektziele:
1. Zusammenfassung von RSS-Blogs generieren und als Mail darstellen
2. Lieferstatus und Trackingnummern von Paketen extrahieren und als Mail darstellen

--> Die Informationsflut soll verringert werden, indem Kunden die relevanten Informationen in Form von Zusammenfassungen gesendet werden.

## Hinweise zum Verwenden:
Nach ziehen des Repos im Ordner rss_mail_summarizer ein .env File erstellen und folgende API Schlüssel hinterlegen:
**GEMINI_API_KEY**: (kann hier erstellt werden: https://ai.google.dev/gemini-api/docs/api-key?hl=de)

**SENDER_PASSWORD**: (Hiermit ist das App Passwort für den Gmail Account gemeint)
